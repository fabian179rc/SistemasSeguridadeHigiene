==============================================================
COMANDOS QUE USARÁS
==============================================================

npm install
  → Instala todas las dependencias (ejecuta UNA SOLA VEZ)
  → Crea la carpeta node_modules/

npm run dev
  → Inicia el servidor de desarrollo
  → Abre http://localhost:3000 en tu navegador
  → Los cambios se actualizan automáticamente

npm run build
  → Crea una carpeta "dist/" lista para subir a Internet

npm run preview
  → Previsualiza cómo se vería la build en producción
